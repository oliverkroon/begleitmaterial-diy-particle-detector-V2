# Begleitmaterial für Workshops zum DIY Teilchendetektor V2

## Anleitungsmaterialien und Messarbeitsblätter zum DIY Teilchendetektor von Rojin Aksu für den Einsatz in Workshops.
Die Begleitmaterialien basieren auf den kürzlich an der Universität Bonn entwickelten DIY Teilchendetektor von Rojin Aksu. Da sich das Detektordesign grundsetzlich vom vorherigen Design von Oliver Keller unterscheidet, gab es noch keine Anleitungsmaterialien für dessen Zusammenbau.\
Diese Lücke soll die Bauanleitung füllen. Außerdem wurde für Workshopzwecke eine Lötanleitung erstellt, wodurch die Teilnehmenden den Umgang mit den Lötkolben üben. Vorherige Löterfahrung wird demnach nicht vorausgesetzt.\
Ein Beispiel wie der Detektor eingesetzt werden kann bieten die Messarbeitsblätter. Diese beinhalten zwei Messungen:
1. Aktivitätsmessungen verschiedener radioaktive Elemente
2. Eigenen Forschungsfragen nachgehen: Hypothesen aufstellen, Durchführung planen, Auswerten, Interpretieren

## Inhalte:
* Zum Download stehen die Materialien im .pdf und .odt Format parat
* Anpassung der Materialien im .odt Format für den eigenen Workshop


## Bekannte Probleme/ Ausstehende Probleme:
Die Hardware auf der Seite von [Rojin Aksu]([https://pages.github.com/](https://github.com/rojinaks/diy-particle-detector-V2)) ist derzeit nur Version 2.4.
Wir arbeiten daran, dass demnächst die Version 2.5 hochgeladen wird, wie sie auch in der Anleitung verwendet wird.

Für das Verwenden der Begleitmaterialien wird noch eine Handreichung für Personen, die einen Workshop selber abhalten wollen, benötigt.
