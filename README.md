# Begleitmaterial für Workshops zum DIY Teilchendetektor V2

## Anleitungsmaterialien und Messarbeitsblätter zum DIY Teilchendetektor von Rojin Aksu für den Einsatz in Workshops.
Diese Begleitmaterialien ermöglichen den Einsatz des neuen Detektordesigns in Workshops. Wer sich für das Bauen einen eigenen Detektors interessiert oder selber einen DIY Detektor Workshop abhalten möchte, kann sich 
Die Begleitmaterialien basieren auf den kürzlich an der Universität Bonn entwickelten DIY Teilchendetektor von Rojin Aksu. Da sich das Detektordesign grundsetzlich von dem Design von Oliver Keller unterscheidet, ...

## Features:
1. Materialien stehen zum Download im .pdf und .odt Format parat 
2. Anpassung der Materialien auf den eigenen Workshop mit dem .odt Format


## Bekannte Probleme:
Die Hardware auf https://github.com/rojinaks/diy-particle-detector-V2 ist die Version 2.4.
Wir arbeiten daran, dass demnächst die Version 2.5 hochgeladen wird, wie sie auch in der Anleitung verwendet wird.

Für das Verwenden der Begleitmaterialien wird noch eine Handreichung für Personen, die einen Workshop selber abhalten wollen, benötigt.
